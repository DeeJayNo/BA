# Ubuntu aufsetzen

#### 1) Ubuntu updaten
```
sudo apt-get update && sudo apt-get upgrade -y
```
#### 2) pip instalieren
```
sudo apt/get install python-pip
```

```
sudo pip3 install jupyterlab
```
